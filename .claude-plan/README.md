# deepinme 프로젝트 계획서

> **한 줄 비전**: "나를 더 잘 설명할 수 있을 때, 우리는 서로를 조금 더 이해할 수 있다."
> 신뢰할 수 있는 심리 검사(평가형)로 시작해, 공유·캐릭터·다양한 테스트(엔터테인먼트)로 확장하고, 궁극적으로 **자기이해 → 심층 분석 → 심리 케어 연결**로 이어지는 하이브리드 플랫폼.

이 디렉토리는 주제별로 계획과 조사를 나눠 기록한다. 각 문서는 독립적으로 갱신 가능.

## 확정된 방향 (2026-07-05 대화 기준)

1. **하이브리드, 단 순서가 있다** — 먼저 **평가형**을 제대로 만들고, 이후 엔터테인먼트 요소를 얹는다.
2. **결과는 일부 무료 공개 + 공유 가능**. 세부 분석·심리 케어 연결은 유료/확장 서비스로.
3. **공유 카드**는 한눈에 이해되게, **귀여운 캐릭터 / 성격에 맞는 애니메이션 캐릭터 매칭**을 함께 노출.
4. **유형별 랜딩페이지 SEO** 채택.
5. **이론 출처 명시**로 정확도·신뢰 확보.
6. **(*)** 에니어그램 **화살표 이론·심리 건강 상태(발달 수준)**를 결과에 반영 검토 → [05](05-psychology-foundation.md).
7. **(#)** 진행 표시는 progress bar + **"단계별 결과 추정치"** 아이디어 → **검사 오염 우려로 지금은 구현하지 않고 옵션으로만 보관** → [04](04-product-ux-data.md).
8. **(&)** word-journey.site처럼 **선택 소요 시간·선택 수정 여부 등 행동 데이터**를 수집해 결과에 반영 → [04](04-product-ux-data.md).
9. **(%)** ingan.ai처럼 **다양한 테스트**를 운영 → [04](04-product-ux-data.md).

> ⚠️ 이 문서군은 **기획 고도(altitude)**에서 작성됨. 코드 레벨 구현 계획은 아직 하지 않는다(사용자 요청). 각 항목은 "무엇을/왜"에 집중하고, "어떻게(구현)"는 팀 합류 후 별도 스펙으로.
