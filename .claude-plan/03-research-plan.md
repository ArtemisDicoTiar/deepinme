## A. 조사 완료 (요약)

### A-1. 벤치마킹 (→ .claude-research 참조)
- **평가형**: 16Personalities(결과 무페이월 + 프리미엄 커리어 리포트, 유형 캐릭터화, 유형별 SEO), Truity(다검사 허브), Enneagram Institute(RHETI 정통성), Crystal(B2B 리드).
- **바이럴형(한국)**: 푸망(평균 35만·최대 169만 플레이, 공유카드·UGC 심테공방·브랜디드 광고), 테스트잇/테몬(무가입 2~3분).

### A-2. ingan.ai (사용자 (%) 레퍼런스) — "다양한 테스트 + 케어 연결" 모델
- 설문의 한계를 넘겠다며 **인스타그램 데이터로 무의식 분석**하는 AI 심리분석. 인스타 ID 입력 → 즉시 리포트.
- **26개 심리 차원** 리포트: 성격, 애착, 연애스타일, 궁합, 위험요인, 성장영역, 관계패턴 등.
- **AI 심리상담 챗** + AI 그림검사로 우울 신호 탐지.
- **시사점**: (1) "다양한 테스트"의 실체 = 하나의 분석을 여러 각도(차원)로 쪼개 보여주기 + 검사 종류 확장. (2) **검사 → 케어(상담) 연결**의 실제 사례 = deepinme Phase 3 참고. (3) 단, SNS 크롤링 방식은 개인정보 리스크 → deepinme는 자체 설문/행동데이터로 시작(→ [01](01-project-plan.md) Scope Out).

### A-3. 행동 데이터 수집 (사용자 (&) word-journey 레퍼런스)
- word-journey.site: 200여 단어를 8단계에 걸쳐 고르고/내려놓는 점진적 필터링. 명시적 행동추적 언급은 없고 결과는 "AI 프롬프트 복사" 방식.
- **학술 근거(수집이 유의미하다는 증거)**:
  - **반응 지연(response latency)**은 태도·기분·성격 측정에서 점증적 신뢰도·타당도를 보임. 정보 접근성(기억에서 얼마나 빨리 떠오르는가)을 반영.
  - 응답 시간은 **부주의·불성실 응답(careless responding) 탐지**에 활용 → 데이터 품질 관리.
  - **게임형 평가(pymetrics 등)**: 반응시간·결정패턴·위험감수·주의전환·오류회복 등 수백 개 행동 데이터 포인트 수집, ML로 특성 추론(평균 25분).
- **시사점**: 반응시간·수정이력은 (1) **성실도 필터**(오염 응답 제거), (2) **확신도/양가성 신호**(오래 망설이거나 자주 바꾼 문항)로 쓸 수 있음. 단, 특성 자체를 바꾸는 근거로 쓰려면 자체 검증 필요.

### A-4. 에니어그램 화살표·심리 건강 상태 (사용자 (*)) — 상세 [05](05-psychology-foundation.md)
- **화살표(통합/분열 방향)**: 성장 시 통합 방향 유형의 건강한 특질을, 스트레스 시 분열 방향 유형의 부정적 특질을 띰. 단 **학계 합의는 약함**.
- **발달 수준(Riso-Hudson Levels of Development)**: 각 유형 10단계, 건강↔불건강 스펙트럼 → "현재 심리 건강 상태" 표현의 이론적 후보.

## B. 조사 백로그 (앞으로 / 담당별)

### B-1. 심리학 담당 (B)
- [ ] Phase 1 검사 선정: 에니어그램 vs MBTI — 각 타당도·신뢰도·라이선스/저작권 비교
- [ ] 공개 사용 가능한 문항 세트/채점 규칙 조사 (저작권 안전한 소스 vs 자체 개발)
- [ ] 화살표·발달수준을 **자기보고 웹검사로 측정 가능한지** (측정 도구 존재? 자체 문항 필요?)

### B-2. 기술·데이터 담당 (A)

- [ ] 기술 스택 후보 비교 (웹 우선 SSR/SEO 친화 프레임워크 등) — 별도 스펙
