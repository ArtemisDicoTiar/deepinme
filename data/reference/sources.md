Header credits the OEPS development URL. This corrected a mis-grouping present in the HTML page summary (e.g., 'I naturally emerge as a leader' keys to Type 8, not Type 3).
- [OSPP Enneagram of Personality Scales v2.0 (ODT scale file)](https://openpsychometrics.org/tests/OEPS/development/OSPP-Enneagram-of-Personality-Scales-v2.odt) — Creative Commons (OSPP project policy; treated as CC BY-NC-SA 4.0) · 54-item version = 36 v1 Likert items + 18 bipolar semantic-differential items (Q37-Q54) with full scoring key including reverse-scored (6-Q) and multi-keyed items. Basis for treating OEPS items as CC-licensed and reusable with attribution.
- [OSPP DISC Assessment (ODAT) — development page (sibling page, license evidence)](https://openpsychometrics.org/tests/ODAT/development/) — CC BY-NC-SA 4.0 (explicitly stated in page footer) · Footer reads: 'All text and data on this page is licensed under a Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International License.' This is the concrete evidence for which CC variant the project uses, applied by inference to the OEPS pages, which lack an explicit footer.
- [OEPS interactive test page](https://openpsychometrics.org/tests/OEPS/) — Educational-use disclaimer; no explicit reuse license on page · Live 54-item test. Page disclaimer: 'free to take and provided for educational use only ... not psychological advice ... without any guarantee of accuracy.' Responses 'may be recorded and anonymously used for research or otherwise distributed.' Project advises against embedding the live test in other apps.
- [Kastelein, K. (2021) — reliability/validity analysis of the OEPS (Olivet Nazarene University honors thesis), summarized on the OEPS dev page](https://openpsychometrics.org/tests/OEPS/development/) — Underlying thesis is copyrighted academic work — do not reproduce; cited for its findings only · Independent evaluation concluded the OEPS 'did not show strong reliability or validity evidence' (e.g., test-retest ~0.68 vs 0.85 for the BFI; low internal consistency). The OEPS author concurs it is a weak psychometric measure.
